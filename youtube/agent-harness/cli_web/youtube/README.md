# cli-web-youtube

> Generated by [CLI-Anything-Web](../../../../cli-anything-web-plugin/) from [youtube.com](https://www.youtube.com)

Search YouTube videos, get video details, browse trending content, explore channels, and fetch video transcripts — all from the command line.

## Installation

```bash
pip install cli-web-youtube
```

## Usage

```bash
# Search for videos
cli-web-youtube search videos "python tutorial" --limit 10

# Get video details
cli-web-youtube video get dQw4w9WgXcQ --json

# Browse trending
cli-web-youtube trending list --category music --limit 10

# Channel info
cli-web-youtube channel get @MrBeast --json

# Video transcript (timestamped, or plain text for piping into an LLM)
cli-web-youtube transcript get dQw4w9WgXcQ
cli-web-youtube transcript get dQw4w9WgXcQ --text-only
cli-web-youtube transcript get dQw4w9WgXcQ --lang en --translate es --json

# List available transcript languages
cli-web-youtube transcript list dQw4w9WgXcQ

# Interactive REPL
cli-web-youtube
```

## Auth

No authentication required — all commands access public YouTube content.

## JSON Output

All commands support `--json` for structured output:

```bash
cli-web-youtube search videos "AI" --limit 3 --json
cli-web-youtube video get dQw4w9WgXcQ --json
```
